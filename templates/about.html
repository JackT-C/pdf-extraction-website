About This Tool

Purpose

This tool automates the extraction of critical data from medical reports, specifically designed for pathologists working with clinical trial data.

Pathologists receive reports from various laboratories containing technical information on patients participating in clinical trials. These reports are often inconsistent in structure, text-heavy, and require expert knowledge to interpret.

Benefits

  • Time Savings: Eliminates manual data extraction
  • Consistency: Standardized data format across reports
  • Accuracy: Reduces human error in data transcription
  • Efficiency: Frees pathologists for high-value work
  • Searchable: Converts image-based reports to text
  • Scalable: Process multiple reports quickly

Key Features

Genetic Report Extraction
  • Disease name and panel information
  • Methodology and nucleic acid details
  • Library prep and platform data
  • Tumour fraction and LOH status
  • Microsatellite instability status
  • Gene mutations and co-occurring results
  • Clinical significance and pathogenicity
  • Variant types and frequencies
  • Patient demographics
  • Assay performance characteristics
IHC Report Extraction
  • Disease name and panel information
  • Tumour type and biopsy location
  • IHC test names (FolR1, PDL1)
  • Clone information
  • Expression scores and percentages
  • Cut-off criteria for interpretation
  • Final positive/negative interpretation
  • Patient demographics

Output Format

The tool generates Excel files with multiple sheets:

  • Genetic_Report: All genetic report data fields
  • IHC_Report: All IHC report data fields
  • Combined_Report: Both report types in one sheet

All missing or unidentifiable data fields are marked as "N/A" for transparency and consistency.
